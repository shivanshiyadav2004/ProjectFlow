## ✨ Features

- User Authentication & Authorization
- JWT-based Login System
- Student Dashboard
- Teacher Dashboard
- Project Submission Workflow
- Project Approval & Rejection
- File Upload Integration
- Cloudinary Storage
- Responsive UI
- MongoDB Atlas Integration

---

## 🛠️ Architecture

Frontend (React + Vite)
↓
Backend API (Node.js + Express)
↓
MongoDB Atlas
↓
Cloudinary Storage
